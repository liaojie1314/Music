# Music
使用java和Kotlin开发的一款音乐app
# 使用的网易云接口  
1.获取歌曲详情  
2.获取用户歌单   
3.获取评论信息  
4.获取用户头像  
5.获取每日推荐  
6.网易云账号登录
# 实现功能
1.搜索音乐       
2.查看歌曲详情    
3.获取本地音乐  
4.记录播放历史    
5.网易云账号登录   
6.播放暂停下一首上一首   
7.列表循环，单曲循环，随机播放  
8.设置（跳过错误音乐，匹配网络歌词，允许移动网络下播放音乐）   
9.收藏音乐（新增有点bug退出界面样式会变）   
10.动态获取权限（新增）    
11.查看评论（新增）
